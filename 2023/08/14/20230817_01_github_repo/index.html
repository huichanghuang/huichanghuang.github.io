3. 授权远程服务器（or本地）连接github（前提是安装了git和ssh）

  1. 配置github用户名和邮箱
  2. rsa方式连接
    1. ssh-keygen创建密钥
    2. 将~/.ssh/id_rsa.pub的内容添加到github创建新的key
    3. ssh -T git@github.com连接测试
    4. 出现Hi xxx! You've successfully authenticated, but GitHub does not provide shell access.则表示连接成功
  3. ecdsa方式连接
    1. ssh-keygen -t ecdsa -b 521 -C "myname@mymail.com"创建密钥
    2. 将~/.ssh/id_ecdsa.pub的内容添加到[github]创建新的key
    3. ssh -T git@github.com连接测试
    4. 出现Hi xxx! You've successfully authenticated, but GitHub does not provide shell access.则表示连接成功
